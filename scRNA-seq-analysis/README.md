# Single-Cell RNA-Seq Analysis Scripts

R scripts for sc-RNA-seq analysis using Seurat pipeline.

## Files

- **UMAP_visualization.R** - UMAP projections of cell clusters
- **Dotplot_visualization.R** - Marker gene expression across cell types

## Key Techniques

✓ Quality control and preprocessing
✓ Dimensionality reduction (PCA, UMAP)
✓ Unsupervised clustering
✓ Differential gene expression
✓ Cell type annotation
